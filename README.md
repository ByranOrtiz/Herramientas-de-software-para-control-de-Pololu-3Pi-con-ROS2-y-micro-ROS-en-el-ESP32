# Herramientas-de-software-para-control-de-Pololu-3Pi-con-ROS2-y-micro-ROS-en-el-ESP32
Este es el repositorio oficial de mi trabajo de graduación titulado: Desarrollo de herramientas de software para control y monitoreo de agentes roboticos Pololu 3Pi+ mediante ROS2 dentro del ecosistema Robotat.

El repositorio cuenta con dos carpetas principales: "ros2_ws" y "MicroRos firmware". ros2_ws es el workspace completo del desarrollo de las herramientas de software para el control del robot. Dentro de ros2_ws se encuentra la carpeta "maps" donde se cargan los mapas generados por el nodo "map_generator_node". Así como la carpeta "logs_multi_pose", donde se guardan los datos de posición en formato CSV cuando se corre el nodo "multi_agent_controller" La carpeta "MicroRos firmware" cuenta con un proyecto de plaftormio ya configurado, dentro de src se encuentra el archivo main.cpp que contiene el firmware micro-ROS para el esp32_devkitc_v4, se incluye el archivo platformio.ini, que es indispensable para la configuración del proyecto de platformio.
